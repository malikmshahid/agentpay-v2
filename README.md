# 🤖 AgentPay
### AI-Powered USDC Payment Agent on Arc

## What is AgentPay?
AgentPay is an autonomous AI agent that can research, negotiate, 
and execute USDC payments on Arc blockchain on behalf of users.

## Tech Stack
- **Frontend:** React + Vite
- **Backend:** Python FastAPI
- **Blockchain:** Arc Testnet
- **Payments:** Circle USDC
- **AI:** Autonomous payment decision engine

## Features
- ⚡ Instant USDC payments on Arc
- 🤖 AI agent decides payment routing
- 📊 Real-time transaction dashboard
- 🔐 Secure Circle API integration

## Setup
### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
